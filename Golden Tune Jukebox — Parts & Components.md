# Golden Tune Jukebox – Parts & Components List
## 1) Power & Distribution
```
1x 5V DC power supply, 8A (minimum) [AliExpress](https://de.aliexpress.com/item/1005007332568357.html)
1x DC input terminal block (2-screw) [AliExpress](https://de.aliexpress.com/item/4001075959057.html)
2x 4-way terminal block [AliExpress](https://de.aliexpress.com/item/1005005622352405.html)
```

## 2) Smoothing & Protection
```
1x Electrolytic capacitor 1000 µF, 6.3 V (or higher) [AliExpress](https://de.aliexpress.com/item/4001042194301.html)
1x 220 Ω resistor (for WS2812B data line) [AliExpress](https://de.aliexpress.com/item/1005004002774546.html)
```
