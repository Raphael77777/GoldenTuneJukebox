# Golden Tune Jukebox – Parts & Components List
## 1) Power & Distribution
```
1x 5V DC power supply, 8A (minimum)
1x DC input terminal block (2-screw)
2x 4-way terminal block
```

## 2) Smoothing & Protection
```
1x Electrolytic capacitor 1000 µF, 6.3 V (or higher)
1x 220 Ω resistor (for WS2812B data line)
```

## 3) Raspberry Pi & Power Injection
```
1x Raspberry Pi
1x USB-C decoy (to feed 5 V from your PSU into Pi USB-C safely)
```

## 4) LED Lighting
```
1x WS2812B 5V addressable RGB LED strip (DC5V, ~1m, 60LEDs)
1x JST-SM 3-pin connectors (pair)
```

## 5) Audio
```
1x USB speaker (plug-and-play with Raspberry Pi USB)
```

## 6) Display
```
1x HDMI display (5” screen)
1x HDMI cable
1x Micro HDMI adapter
1x USB cable (for touch + power)
```

## 7) Rotary Encoder
```
1x Rotary encoder with push-button (KY-040)
```

## 8) Power Switch
```
1x Toggle Switch
```

## 9) Wire
```
- Wire 18awg
- Wire 24awg
- Pin Wire Connector
- Heat shrink tubes
```

## 10) Screw
```
4x Standoffs M2.5
4x M2.5x4mm
20x M4x8mm
```

# Tools (if you don’t already have them)
```
- Wire stripper & cutter
- Crimper / pliers
- Soldering iron + solder
- Multimeter
- Small screwdriver set
```
